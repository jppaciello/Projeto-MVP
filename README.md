# Projeto-MVP
Projeto MVP - Ciência de Dados e Analytics
🎯 Objetivo

Construir e avaliar modelos de Machine Learning capazes de classificar o nível de queima calórica de uma sessão de exercício, comparando diferentes abordagens e identificando as variáveis mais relevantes para o gasto energético.

Aplicações práticas:


Apps de fitness que classificam sessões automaticamente
Profissionais de educação física que precisam triar praticantes
Sistemas de recomendação de treino personalizados
